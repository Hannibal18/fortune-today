
# Blueprint: 오늘의 운세 웹사이트

## 개요

이 문서는 사용자의 요청에 따라 구축된 "오늘의 운세" 웹사이트의 청사진입니다. 이 웹사이트는 사용자에게 무작위로 생성된 운세를 보여주는 간단한 대화형 애플리케이션입니다.

## 기능

- **운세 보기 버튼:** 사용자가 클릭하여 오늘의 운세를 요청할 수 있습니다.
- **운세 표시:** 버튼을 클릭하면 화면에 무작위 운세가 표시됩니다.
- **다크모드 전환:** 🌙/☀️ 버튼을 눌러 테마를 전환하고, 사용자의 선택을 로컬 저장소에 저장하여 다시 방문 시에도 유지합니다.

## 디자인 및 스타일

- **레이아웃:** 중앙 정렬된 콘텐츠와 함께 간단하고 명확한 레이아웃을 사용합니다.
- **색상 팔레트:** 편안하고 신비로운 느낌을 주는 색상 조합을 사용합니다.
- **글꼴:** 가독성이 좋은 글꼴을 선택합니다.
- **애니메이션:** 운세가 부드럽게 나타나는 페이드 인 효과를 추가합니다.

## 구현 계획

1.  **`index.html` 수정:**
    *   페이지 제목을 "오늘의 운세"로 변경합니다.
    *   운세 앱을 위한 기본 HTML 구조를 설정합니다. 여기에는 제목, 운세 보기 버튼, 운세가 표시될 영역이 포함됩니다.
2.  **`style.css` 수정:**
    *   페이지의 전반적인 스타일(배경, 글꼴, 색상)을 정의합니다.
    *   버튼과 운세 표시에 스타일을 적용합니다.
    *   반응형 디자인을 위해 미디어 쿼리를 추가합니다.
3.  **`main.js` 수정:**
    *   운세 목록을 포함하는 배열을 만듭니다.
    *   "운세 보기" 버튼에 클릭 이벤트 리스너를 추가합니다.
    *   클릭 시 운세 배열에서 무작위로 하나를 선택하여 화면에 표시하는 함수를 구현합니다.
